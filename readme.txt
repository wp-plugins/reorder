== Description ==

Enables simple drag and drop reordering of all post types. This is like the "postMash" plugin, but works with 3.0.

This plugin only support reordering of top-level posts (no hierarchy).

== Installation ==

1. Upload `reorder.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done

== Screenshots ==
1. A "Reorder" links automatically gets added under each custom post type.
2. List of top-level posts.
3. Drag and drop to reorder.
